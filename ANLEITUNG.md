# Leadinfo-HTML-Dateien

In allen enthaltenen HTML-Dateien ist der Leadinfo-Trackingcode mit der ID

LI-5FD0DEA7F008F

direkt vor dem schließenden </head>-Tag eingebunden.

## Einbau in GitHub

Für jede Datei im Repository:

1. Datei öffnen.
2. Auf das Stiftsymbol „Edit this file“ klicken.
3. Gesamten bisherigen Inhalt markieren und löschen.
4. Den kompletten Inhalt der gleichnamigen Datei aus diesem Paket einsetzen.
5. „Commit changes“ ausführen.

Empfohlene Commit-Nachricht:

Add Leadinfo tracking to all pages

Wichtig: Die Dateinamen unverändert lassen.
