# neckarfreunde Website — Release 1.0

Offizielle Website der neckarfreunde Strategieberatung aus Stuttgart.

## Struktur
- `index.html` — Startseite
- `buero.html` — Team & Netzwerk
- `zusammenarbeit.html` — Service-Formate
- `sprinttag.html` — Landingpage Sprinttag
- `arbeiten.html` — Case Studies
- `kunden.html` — Kundenlogos & Bewertungen
- `perspektiven.html` — Artikel & Gedanken
- `perspektive-*.html` — Einzelne Artikel
- `kontakt.html` — Kontaktformular
- `impressum.html` — Legal
- `datenschutz.html` — DSGVO + Leadinfo
- `404.html` — Error Page

## Assets
- `assets/style.css` — Design System
- `assets/script.js` — Client-side Logic (Leadinfo)
- `assets/logo.svg` — Brand SVG
- `assets/images/` — Fotos, Logos, Awards

## Deployment
### GitHub Pages
```bash
git push origin main
```

### IONOS
1. CNAME auf www.neckarfreunde.de setzen
2. Dateien hochladen
3. HTTPS aktivieren
